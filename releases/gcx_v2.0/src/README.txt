************************************************************
***************** THE GCX XQUERY ENGINE ********************
************************************************************
GCX is a streaming XQuery engine. It has originally been
developed as part of a research project at the Saarland
University Database Group and is currently continued at
Freiburg University. You can find more information about
GCX at the project page

	http://www-db.cs.uni-sb.de/projects/streams/gcx/
